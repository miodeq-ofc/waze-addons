## PL – Polski

**WME Addons** to kolekcja dodatków i usprawnień do edytora map Waze (WME), które ułatwiają korzystanie z map oraz integrację z innymi skryptami użytkowników.

### Funkcje
- Dodawanie nowych narzędzi i opcji do edytora WME
- Wspieranie integracji z innymi skryptami użytkowników

### Instalacja
1. Zainstaluj [Tampermonkey](https://www.tampermonkey.net/) w swojej przeglądarce.
2. Kliknij w plik (script.user.js) w repozytorium → Raw
3. Następnie w Tampermonkey kliknij → Zainstaluj
4. Gotowe

### Aktualizacje
Skrypty są automatycznie aktualizowane.



## EN – English

**WME Addons** is a collection of add-ons and enhancements for the Waze Map Editor (WME) that make map editing easier and allow integration with other user scripts.

### Features
- Add new tools and options to the WME editor
- Support integration with other user scripts

### Installation
1. Install [Tampermonkey](https://www.tampermonkey.net/) in your browser.
2. Click on the script file (script.user.js) in the repository → Raw
3. Then in Tampermonkey, click → Install
4. Done

### Updates
Scripts are automatically updated.
